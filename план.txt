Создать телефонный справочник с
возможностью импорта и экспорта данных в
формате .txt. Фамилия, имя, отчество, номер
телефона - данные, которые должны находиться
в файле.


0. Сформулировать формат данных


1. Программа должна выводить данные
    1.0 Запрос формата вывода данных (1.1 или 1.2)
        1.0.0 Вывод всех данных
        1.0.1 Введите фильтр
        1.0.2 Вывод отфильтрованных данных
    1.1 Написать код по выводу всех данных в консоль
    1.2 Вывод данных по фильтру


2. Программа должна сохранять данные в текстовом файле
    2.1 Определить способ хранения данных
        2.1.0 Списки строк
        2.1.1 Списки словарей (! сложно)
        2.1.2 Сохранение данных построчно, без переформатирования
    2.2 Формат хранения данных


3. Пользователь может ввести одну и
характеристик для поиска определенной
записи(Например имя или фамилию человека)
    3.1 Чтение сохраненных данных
    3.2 Поиск по сохраненным данным
    3.3 Фильтр вывода (1.2)
4. Использование функций. Ваша
программа не должна быть линейной


5. Изменение данных
